# context-harness

`context-harness` is a local-first framework for building a personal AI context loop.

V1 focuses on two local Code Agent sources:

- Codex
- Claude Code

The project keeps mechanism and user data separate. The repository contains the CLI, skills, hook installers, templates, docs, and tests. Conversations, memory, logs, and sync state live in a configurable data directory such as `~/.context-harness`.
